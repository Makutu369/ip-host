# ip - host
Api implementation of ip to host using maxmind.com geopip database

## installation 
1. clone the repository
   ```
   git clone https://github.com/Makutu369/ip-host.git
   ```
3. install the dependencies
   at the root of the project folder
   ```
   npm install
   ```
     
   
4. run the program
   ```
   node index.js
   ```
